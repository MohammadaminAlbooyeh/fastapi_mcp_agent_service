# Agents

## Query Agent
- **Name:** `query`
- **Purpose:** Execute database queries
- **Tools:** database_tool

## Data Processor Agent
- **Name:** `processor`
- **Purpose:** Data transformation and analysis
- **Tools:** file_tool, calculator_tool

## Research Agent
- **Name:** `research`
- **Purpose:** Information gathering and research
- **Tools:** search_tool, api_tool

## Orchestrator
Routes requests to the appropriate agent and manages execution.
